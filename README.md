# 🌐 Multi Nginx Project (2 Sites)

This project is a simple Nginx setup that serves two websites on the same Linux server.

---

## 📁 Project Structure

multi-nginx/
├── sites/
│   ├── site1/
│   │   └── index.html
│   └── site2/
│       └── index.html
│
├── nginx-config/
│   ├── site1.conf
│   └── site2.conf
│
└── README.md

---

## 🚀 What the project does

- Runs two websites on the same Nginx server
- Site 1 is accessible on port 80
- Site 2 is accessible on port 8080
- Separate configuration for each site
- Manual deployment on a Linux server

---

## ⚙️ How to access the sites

### 🌐 Site 1
http://SERVER_IP/

### 🌐 Site 2
http://SERVER_IP:8080/
